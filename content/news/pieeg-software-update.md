We're excited to announce a major software update for all PiEEG devices! This release represents months of development work and incorporates feedback from hundreds of researchers worldwide.

## What's New

### Enhanced Signal Processing
- Improved noise reduction algorithms
- Real-time artifact detection and removal
- Advanced filtering options with customizable parameters
- Support for multiple reference electrode configurations

### New Analysis Features
- Built-in FFT and spectral analysis tools
- Time-frequency analysis (wavelet transforms)
- Connectivity analysis (coherence, phase synchronization)
- Automated peak detection and event-related potential analysis

### User Interface Improvements
- Modern, intuitive dashboard design
- Real-time signal visualization with multiple viewing modes
- Customizable layouts and workspaces
- Dark mode support

### Performance Enhancements
- 50% faster data acquisition
- Reduced latency for real-time applications
- Better memory management for long recording sessions
- Optimized for both desktop and mobile devices

## Compatibility

This update is compatible with:
- PiEEG-16
- PiEEG-24
- PiEEG-32
- IronBCI (all variants)
- MicroBCI with STM32 Shield

## Download & Installation

The new software is available for download from our [official repository](https://github.com/pieeg-club). Installation instructions and migration guides are provided in the documentation.

For users with automatic updates enabled, the software will update automatically within the next 48 hours.
